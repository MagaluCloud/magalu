# Cloud Events API Product.

## Usage:
```bash
Usage:
  ./mgc events [flags]
  ./mgc events [command]
```

## Product catalog:
- Commands:
- events      Endpoints related to listing cloud events emitted by other products.

## Other commands:
- Flags:
- -h, --help      help for events
- -v, --version   version for events

Global Flags:
