# Set the active Tenant to be used for all subsequential requests

## Usage:
```bash
Usage:
  ./mgc auth tenant set [uuid] [flags]
```

## Product catalog:
- Flags:
- -h, --help          help for set
- --uuid string   The UUID of the desired Tenant. To list all possible IDs, run auth tenant list (required)
