Returns a list of all volume types from db

Usage:
  ./cli block-storage volume-types list-all [flags]
  ./cli block-storage volume-types list-all [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help      help for list-all
  -v, --version   version for list-all

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli block-storage volume-types list-all [command] --help" for more information about a command.
