Block Storage Volume Attachment

Usage:
  ./cli block-storage volume-attach-blueprint [flags]
  ./cli block-storage volume-attach-blueprint [command]

Commands:
  create      Attach a volume to a virtual machine instance
  delete      Detach a volume from a virtual machine instance
  get         Check if a volume is attached to a virtual machine instance
  update      Update a block storage volume attachment

Flags:
  -h, --help   help for volume-attach-blueprint

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli block-storage volume-attach-blueprint [command] --help" for more information about a command.
