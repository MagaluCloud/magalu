Attach a volume to a virtual machine instance

Usage:
  ./cli block-storage volume-attach-blueprint create [flags]
  ./cli block-storage volume-attach-blueprint create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --block-storage-id uuid     Volume Id: id
  -h, --help                      help for create
      --virtual-machine-id uuid   Virtual Machine Id

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli block-storage volume-attach-blueprint create [command] --help" for more information about a command.
