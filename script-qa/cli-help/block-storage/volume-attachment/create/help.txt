Attach a volume to a virtual machine instance

Usage:
  ./cli block-storage volume-attachment create [flags]
  ./cli block-storage volume-attachment create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help                        help for create
      --id string                   Block storage volume ID to be attached
      --virtual-machine-id string   ID of the virtual machine instance to attach the volume

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli block-storage volume-attachment create [command] --help" for more information about a command.
