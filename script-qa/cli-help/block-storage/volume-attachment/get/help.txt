Check if a volume is attached to a virtual machine instance

Usage:
  ./cli block-storage volume-attachment get [flags]

Flags:
  -h, --help                        help for get
      --id string                   Block storage volume ID (required)
      --virtual-machine-id string   Instance ID of the virtual machine to which the volume is attached (required)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
