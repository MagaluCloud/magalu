Check if a volume is attached to a virtual machine instance

Usage:
  ./cli block-storage volume-attachment get [flags]

Flags:
  -h, --help                        help for get
      --id string                   Block storage volume ID
      --virtual-machine-id string   Instance ID of the virtual machine to which the volume is attached

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
