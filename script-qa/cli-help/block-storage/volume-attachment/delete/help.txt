Detach a volume from a virtual machine instance

Usage:
  ./cli block-storage volume-attachment delete [flags]
  ./cli block-storage volume-attachment delete [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help                        help for delete
      --id string                   Block storage volume ID to be detached
      --virtual-machine-id string   ID of the virtual machine instance to detach the volume from

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli block-storage volume-attachment delete [command] --help" for more information about a command.
