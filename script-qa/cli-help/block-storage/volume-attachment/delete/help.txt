Detach a volume from a virtual machine instance

Usage:
  ./cli block-storage volume-attachment delete [flags]

Flags:
  -h, --help                        help for delete
      --id string                   Block storage volume ID to be detached
      --virtual-machine-id string   ID of the virtual machine instance to detach the volume from

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
