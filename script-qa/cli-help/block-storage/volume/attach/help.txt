Attach a volume into an instance, both from the provided tenant_id

Usage:
  ./cli block-storage volume attach [id] [virtual-machine-id] [flags]
  ./cli block-storage volume attach [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help                      help for attach
      --id uuid                   Volume Id: id
  -v, --version                   version for attach
      --virtual-machine-id uuid   Virtual Machine Id: virtual_machine_id

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli block-storage volume attach [command] --help" for more information about a command.
