  ./cli block-storage usage [flags]

Commands:
  list        Get Usage

Flags:

  -v, --version   version for usage

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli block-storage usage [command] --help" for more information about a command.
