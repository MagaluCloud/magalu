Delete a security group from the provided tenant_id

Usage:
  ./cli network security-group delete [security-group-id] [flags]

Flags:
  -h, --help                       help for delete
      --security-group-id string   Security Group Id (required)
  -v, --version                    version for delete

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-mgl-1", "br-ne-1" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
