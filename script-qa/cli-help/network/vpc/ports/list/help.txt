List VPC ports

Usage:
  ./cli network vpc ports list [vpc-id] [flags]

Flags:
      --control.limit integer        Items Per Page (min: 1) (default 10)
      --control.offset integer       Page Number (min: 1) (default 1)
      --detailed                     Detailed (default true)
  -h, --help                         help for list
      --port-id-list array(string)   Port Id List (default [])
  -v, --version                      version for list
      --vpc-id string                Vpc Id (required)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
