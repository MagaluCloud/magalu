Update a public ip from the provided tenant_id

Usage:
  ./cli network worker-public-ips update [public-ip-id] [flags]

Flags:
      --error anyValue         Error
                               Use --error=help for more details
      --external-id anyValue   External Id
                               Use --external-id=help for more details
  -h, --help                   help for update
      --public-ip anyValue     Public Ip
                               Use --public-ip=help for more details
      --public-ip-id string    Public Ip Id (required)
      --status enum            PublicIpStatus (one of "created", "error" or "pending") (required)
  -v, --version                version for update

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-mgl1", "br-ne1" or "br-se1") (default "br-ne1")
      --server-url uri         Manually specify the server to use
