Create a Rule

Usage:
  ./cli network rule security-groups-rules create [security-group-id] [flags]
  ./cli network rule security-groups-rules create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --direction string           
      --ethertype string           
  -h, --help                       help for create
      --port-range-max integer     
      --port-range-min integer     
      --protocol string            
      --remote-group-id string     
      --remote-ip-prefix string    
      --security-group-id string   
      --validate-quota              (default true)
  -v, --version                    version for create
      --wait                        (default true)
      --wait-timeout integer       the approximate time in seconds you want to wait when simulating the request as synchronous (only works with wait=true) (default 120)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli network rule security-groups-rules create [command] --help" for more information about a command.
