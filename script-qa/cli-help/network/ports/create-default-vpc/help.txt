Create a Port on the default VPC

Usage:
  ./cli network ports create-default-vpc [flags]

Flags:
      --cli.list-links enum[=table]   List all available links for this command (one of "json", "table" or "yaml")
      --has-pip anyValue              Has Pip
                                      Use --has-pip=help for more details (default true)
      --has-sg anyValue               Has Sg
                                      Use --has-sg=help for more details (default true)
  -h, --help                          help for create-default-vpc
      --name string                   Name (between 5 and 100 characters) (required)
      --security-groups-id anyValue   Security Groups Id
                                      Use --security-groups-id=help for more details (default [])
      --subnets anyValue              Subnets
                                      Use --subnets=help for more details (default [])
  -v, --version                       version for create-default-vpc

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-mgl1", "br-ne1" or "br-se1") (default "br-ne1")
      --server-url uri         Manually specify the server to use
