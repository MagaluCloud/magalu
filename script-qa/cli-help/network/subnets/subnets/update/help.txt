Update a subnet from the provided tenant_id

Usage:
  ./cli network subnets subnets update [subnet-id] [flags]
  ./cli network subnets subnets update [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --dns-nameservers array(ipvanyaddress)    (default [])
  -h, --help                                   help for update
      --subnet-id string                       
  -v, --version                                version for update

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli network subnets subnets update [command] --help" for more information about a command.
