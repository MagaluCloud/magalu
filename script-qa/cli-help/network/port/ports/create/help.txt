Create a Port on the default VPC

Usage:
  ./cli network port ports create [flags]
  ./cli network port ports create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --has-pip                             (default true)
      --has-sg                              (default true)
  -h, --help                               help for create
      --name string                        (between 5 and 100 characters)
      --security-groups-id array(string)    (default [])
      --subnets array(string)               (default [])
  -v, --version                            version for create

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli network port ports create [command] --help" for more information about a command.
