List VPC ports

Usage:
  ./cli network port vpcs-ports list [vpc] [flags]
  ./cli network port vpcs-ports list [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      ---limit integer               Items Per Page (min: 1) (default 10)
      ---offset integer              Page Number (min: 1) (default 1)
      --detailed                     Detailed (default true)
  -h, --help                         help for list
      --port-id-list array(string)   Port Id List (default [])
  -v, --version                      version for list
      --vpc string                   Vpc Id

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli network port vpcs-ports list [command] --help" for more information about a command.
