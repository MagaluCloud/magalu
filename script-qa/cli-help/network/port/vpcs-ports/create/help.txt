Create a Port

Usage:
  ./cli network port vpcs-ports create [vpc-id] [flags]
  ./cli network port vpcs-ports create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --has-pip                             (default true)
      --has-sg                              (default true)
  -h, --help                               help for create
      --name string                        (between 5 and 100 characters)
      --security-groups-id array(string)    (default [])
      --subnets array(string)               (default [])
  -v, --version                            version for create
      --vpc-id string

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli network port vpcs-ports create [command] --help" for more information about a command.
