Create a Public IP

Usage:
  ./cli network public-ip vpcs-public-ips create [vpc-id] [flags]
  ./cli network public-ip vpcs-public-ips create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --description string   
  -h, --help                 help for create
      --validate-quota        (default true)
  -v, --version              version for create
      --vpc-id string

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli network public-ip vpcs-public-ips create [command] --help" for more information about a command.
