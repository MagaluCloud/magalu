APIs referentes ao produto de VPC

Usage:
  ./mgc network [flags]
  ./mgc network [command]

Commands:
  ports           VPC Port
  public-ips      VPC Public IPs
  rules           VPC Rules
  security-groups VPC Security Groups
  subnets         VPC Subnets
  vpcs            Virtual Private Cloud (VPC)

Flags:
  -h, --help      help for network
  -v, --version   version for network

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./mgc network [command] --help" for more information about a command.
