Create a Public IP

Usage:
  ./cli network vpcs public-ips create [vpc-id] [flags]

Flags:
      --cli.list-links enum[=table]   List all available links for this command (one of "json", "table" or "yaml")
      --description anyValue          Description
                                      Use --description=help for more details (default "")
  -h, --help                          help for create
      --validate-quota                validateQuota (default true)
  -v, --version                       version for create
      --vpc-id string                 Vpc Id (required)
      --wait                          wait (default true)
      --wait-timeout string           waitTimeout: the approximate time in seconds you want to wait when simulating the request as synchronous (only works with wait=true) (default "120")

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-mgl1", "br-ne1" or "br-se1") (default "br-ne1")
      --server-url uri         Manually specify the server to use
