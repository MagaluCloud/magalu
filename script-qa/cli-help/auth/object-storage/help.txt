Object Storage uses a different set of credentials when compared
to normal HTTP requests. Two keys are needed, the 'SecretKey' and
'AccessKeyId'. Instructions on how to create these can be found
here: https://id.magalu.com/api-keys

Usage:
  ./cli auth object-storage [flags]
  ./cli auth object-storage [command]

Available Commands:
  get         Get the current Object Storage credentials
  set         Set the credentials values used for Object Storage requests

Flags:
  -h, --help   help for object-storage

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli auth object-storage [command] --help" for more information about a command.
