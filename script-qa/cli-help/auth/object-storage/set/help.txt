Set the credentials values used for Object Storage requests

Usage:
  ./cli auth object-storage set [access-key-id] [secret-access-key] [flags]

Flags:
      --access-key-id string       Access key id value
  -h, --help                       help for set
      --secret-access-key string   Secret access key value

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
