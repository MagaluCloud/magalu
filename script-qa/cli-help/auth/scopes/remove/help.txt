Remove scopes from the current scopes in the access token.
Run 'auth scopes list-current' to see current scopes

Usage:
  ./cli auth scopes remove [scopes...] [flags]

Flags:
  -h, --help                   help for remove
      --scopes array(string)   Scopes to be removed from the current access token (required)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
