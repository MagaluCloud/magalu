Set the scopes for the current scopes in the access token.
Run 'auth scopes list-all' to see all available scopes

Usage:
  ./cli auth scopes set [scopes...] [flags]

Flags:
  -h, --help                   help for set
      --scopes array(string)   The new scopes to be saved in the current access token (required)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
