Add new scopes to the current access token. Run 'auth scopes list-all'
to see all available scopes to be added

Usage:
  ./cli auth scopes add [scopes...] [flags]

Flags:
  -h, --help                   help for add
      --scopes array(string)   Scopes to be added to the current access token (required)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
