Routes related to Container Registry management.

Usage:
  ./cli mcr registries [flags]
  ./cli mcr registries [command]

Commands:
  create      Create a container registry
  credentials Get credentials for container registry
  delete      Delete a container registry by registry_uuid
  get         Get registry information
  list        List all container registries

Flags:
  -h, --help      help for registries
  -v, --version   version for registries

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli mcr registries [command] --help" for more information about a command.
