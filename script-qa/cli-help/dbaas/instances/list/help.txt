Returns a list of database instances for a x-tenant-id.

Usage:
  ./cli dbaas instances list [flags]
  ./cli dbaas instances list [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      ---expand _expand=value1,value2   Instance extra attributes or relations to show with the main query. When available, more than one value can be informed using commas. e.g: _expand=value1,value2 (one of replicas)
      ---limit integer                  limit (range: 1 - 25) (default 10)
      ---offset integer                 offset (min: 0)
  -h, --help                            help for list
      --status enum                     Value referring to instace status. (one of ACTIVE|BACKING_UP|CREATING|DELETED|DELETING|ERROR|ERROR_DELETING|PENDING|REBOOT|RESIZING|RESTORING|STARTING|STOPPED|STOPPING)
  -v, --version                         version for list

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli dbaas instances list [command] --help" for more information about a command.
