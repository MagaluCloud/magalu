Returns a list of database instances for a tenant.

Usage:
  ./cli dbaas instances list [flags]

Examples:
  ./cli dbaas instances list --status="ACTIVE"

Flags:
      --expand enum      Instance extra attributes or relations to show with the main query. When available, more than one value can be informed using commas. e.g: '_expand=value1,value2' (must be "replicas")
  -h, --help             help for list
      --limit integer    limit (range: 1 - 25) (default 10)
      --offset integer   offset (min: 0)
      --status enum      Value referring to instace status. (one of "ACTIVE", "BACKING_UP", "CREATING", "DELETED", "DELETING", "ERROR", "ERROR_DELETING", "PENDING", "REBOOT", "RESIZING", "RESTORING", "STARTING", "STOPPED" or "STOPPING")
  -v, --version          version for list

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-mgl-1", "br-ne-1" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
