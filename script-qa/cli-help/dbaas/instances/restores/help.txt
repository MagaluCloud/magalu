Restores a backup for an instance asynchronously.

Usage:
  ./cli dbaas instances restores [id] [flags]
  ./cli dbaas instances restores [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --backup-id uuid   Backup Id
  -h, --help             help for restores
      --id uuid          Instance Id: Value referring to instance Id.
  -v, --version          version for restores

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli dbaas instances restores [command] --help" for more information about a command.
