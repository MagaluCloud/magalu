Returns a database instance detail.

Usage:
  ./cli dbaas instances get [id] [flags]
  ./cli dbaas instances get [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      ---expand _expand=value1,value2   Instance extra attributes or relations to show with the main query. When available, more than one value can be informed using commas. e.g: _expand=value1,value2 (must be "replicas")
  -h, --help                            help for get
      --id uuid                         Database Instance Id: Database Instance Unique Id.
  -v, --version                         version for get

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli dbaas instances get [command] --help" for more information about a command.
