Resizes a database instance.

Usage:
  ./cli dbaas instances resize [id] [flags]
  ./cli dbaas instances resize [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --flavor-id uuid   Flavor Id
  -h, --help             help for resize
      --id uuid          Database Instance Id: Database Instance Unique Id.
  -v, --version          version for resize
      --volume object    Instance Volume Request (single property: size)
                         Use --volume=help for more details

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli dbaas instances resize [command] --help" for more information about a command.
