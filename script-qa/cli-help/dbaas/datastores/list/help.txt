Returns a list of available datastores. A datastore is a combination of engine and version.

Usage:
  ./cli dbaas datastores list [flags]
  ./cli dbaas datastores list [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      ---limit integer    limit (range: 1 - 25) (default 10)
      ---offset integer   offset (min: 0)
  -h, --help              help for list
      --status enum       Datastore Status: Value referring to datastore status. (one of "ACTIVE" or "DEPRECATED")
  -v, --version           version for list

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli dbaas datastores list [command] --help" for more information about a command.
