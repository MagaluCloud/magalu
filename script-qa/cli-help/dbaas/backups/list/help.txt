List all backups.

Usage:
  ./cli dbaas backups list [flags]
  ./cli dbaas backups list [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      ---limit integer    limit (range: 1 - 25) (default 10)
      ---offset integer   offset (min: 0)
  -h, --help              help for list
      --mode enum         Backup Mode: Value referring to backup mode. (one of "FULL" or "INCREMENTAL")
      --status enum       Backup Status: Value referring to backup status. (one of "CREATED", "CREATING", "DELETED", "DELETING", "ERROR" or "PENDING")
      --type enum         Backup Type: Value referring to backup type. (one of "AUTOMATED" or "ON_DEMAND")
  -v, --version           version for list

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli dbaas backups list [command] --help" for more information about a command.
