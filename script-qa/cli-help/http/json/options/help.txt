Call using HTTP OPTIONS

Usage:
  ./cli http json options [flags]
  ./cli http json options [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --body anyValue            Use --body=help for more details
      --headers object           Use --headers=help for more details
  -h, --help                     help for options
      --security array(object)   Use --security=help for more details
      --url string               Golang template with the URL

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli http json options [command] --help" for more information about a command.
