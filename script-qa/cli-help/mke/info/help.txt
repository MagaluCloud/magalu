Rotas relacionadas a informações de configurações do mke-api.

Usage:
  ./cli mke info [flags]
  ./cli mke info [command]

Commands:
  flavors     Lists all available flavors
  versions    Lists all available versios

Flags:
  -h, --help      help for info
  -v, --version   version for info

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli mke info [command] --help" for more information about a command.
