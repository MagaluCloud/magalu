Atualiza nodes de um node pool

Usage:
  ./cli mke nodepool update [cluster-id] [node-pool-id] [flags]
  ./cli mke nodepool update [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --auto-scale object   Objeto que define propriedades para atualização de recursos da carga de trabalho do cluster kubernetes ({"enabled": boolean (default false), "max_replicas": integer (min: 1) (default 1), "min_replicas": integer (min: 1) (default 1)})
      --cluster-id uuid     Identificador no padrão uuid para identificar o cluster
  -h, --help                help for update
      --node-pool-id uuid   Identificador no padrão uuid para identificar o nodepool
      --replicas integer    Quantidade de réplicas dos nós do nodepool
  -v, --version             version for update

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli mke nodepool update [command] --help" for more information about a command.
