Rotas relacionadas a api de Cluster.

Usage:
  ./cli mke cluster [flags]
  ./cli mke cluster [command]

Available Commands:
  create      Create a cluster
  delete      Delete a cluster
  get         List a cluster by id
  kubeconfig  Get kubeconfig cluster
  list        List all clusters

Flags:
  -h, --help      help for cluster
  -v, --version   version for cluster

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli mke cluster [command] --help" for more information about a command.
