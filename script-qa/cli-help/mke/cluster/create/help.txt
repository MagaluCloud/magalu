Criação de um cluster kubernetes

Usage:
  ./cli mke cluster create [flags]
  ./cli mke cluster create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --description string         Descrição do cluster
      --enabled-bastion            Habilita bastion host
      --enabled-server-group       Habilita o uso de server group, com política de anti-affinity, na criação do cluster e dos node pools (default true)
  -h, --help                       help for create
      --name string                Nome do cluster kubernetes. O nome destina-se principalmente à idempotência, e deve ser exclusivo em um namespace. 0 nome não pode ser alterado.
                                   O nome deve seguir as seguintes regras:
                                     - deve conter no máximo 63 caracteres
                                     - deve conter apenas caracteres alfanuméricos minúsculos ou '-'
                                     - deve começar com um caractere alfabético
                                     - deve terminar com um caractere alfanumérico
                                   
      --node-pools array(object)   Conjunto de nodes em um cluster kubernetes ([{"auto_scale": {"enabled": boolean (default false), "max_replicas": integer (min: 1) (default 1), "min_replicas": integer (min: 1) (default 1)}, "flavor": string (one of cloud-k8s.gp1.large|cloud-k8s.gp1.small), "name": string, "replicas": integer (default 1), "tags": [string], "taints": [{"effect": string (one of NoExecute|NoSchedule|PreferNoSchedule), "key": string, "value": string}]}])
      --version enum               Versão do kubernetes nativa do cluster (one of v1.22.9|v1.24.10|v1.26.1|v1.28.2) (default v1.28.2)

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli mke cluster create [command] --help" for more information about a command.
