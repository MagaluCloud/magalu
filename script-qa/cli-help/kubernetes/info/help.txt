Endpoints related to configuration information for the MKE API.

Usage:
  ./cli kubernetes info [flags]
  ./cli kubernetes info [command]

Commands:
  flavors     Lists all available flavors
  versions    Lists all available versions

Flags:
  -h, --help      help for info
  -v, --version   version for info

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli kubernetes info [command] --help" for more information about a command.
