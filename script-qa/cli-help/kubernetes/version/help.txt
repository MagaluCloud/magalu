Endpoints related to configuration information for Kubernetes versions.

Usage:
  ./cli kubernetes version [flags]
  ./cli kubernetes version [command]

Commands:
  list        Lists all available versions

Flags:
  -h, --help      help for version
  -v, --version   version for version

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli kubernetes version [command] --help" for more information about a command.
