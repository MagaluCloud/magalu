Rotas relacionadas a informações de configurações dos flavors do openstack.

Usage:
  ./cli kubernetes flavor [flags]
  ./cli kubernetes flavor [command]

Commands:
  list        Lists all available flavors

Flags:
  -h, --help      help for flavor
  -v, --version   version for flavor

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli kubernetes flavor [command] --help" for more information about a command.
