APIs referentes ao produto de kubernetes

Usage:
  ./cli kubernetes [flags]
  ./cli kubernetes [command]

Commands:
  cluster     Rotas relacionadas a api de cluster.
  flavor      Rotas relacionadas a informações de configurações dos flavors do openstack.
  info        Rotas relacionadas a informações de configurações do mke-api.
  nodepool    Rotas relacionadas a api de Nodepool dentro da api de cluster.
  version     Rotas relacionadas a informações de configurações das versões de kubernetes.

Flags:
  -h, --help      help for kubernetes
  -v, --version   version for kubernetes

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli kubernetes [command] --help" for more information about a command.
