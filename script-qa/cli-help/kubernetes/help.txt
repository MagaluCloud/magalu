APIs related to the Kubernetes product.

Usage:
  ./cli kubernetes [flags]
  ./cli kubernetes [command]

Commands:
  cluster     Endpoints related to the Cluster API.
  flavor      Endpoints related to configuration information for OpenStack flavors.
  info        Endpoints related to configuration information for the MKE API.
  nodepool    Endpoints related to the Nodepool API within the Cluster API.
  version     Endpoints related to configuration information for Kubernetes versions.

Flags:
  -h, --help      help for kubernetes
  -v, --version   version for kubernetes

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli kubernetes [command] --help" for more information about a command.
