Operations with snapshots for instances.

Usage:
  ./cli virtual-machine snapshots [flags]
  ./cli virtual-machine snapshots [command]

Commands:
  latest-snapshots latest-snapshots
  v0-snapshots     v0-snapshots
  v1-snapshots     v1-snapshots

Flags:
  -h, --help      help for snapshots
  -v, --version   version for snapshots

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli virtual-machine snapshots [command] --help" for more information about a command.
