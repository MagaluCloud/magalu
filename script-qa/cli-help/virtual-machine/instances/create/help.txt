Create a Virtual Machine instance

Usage:
  ./cli virtual-machine instances create [flags]
  ./cli virtual-machine instances create [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --allocate-fip                        (default true)
      --availability-zone string           
  -h, --help                               help for create
      --image string                       
      --key-name string                    
      --name string                        
      --network-interfaces array(object)   ([{"id": string}])
      --type string                        
      --user-data string                   
  -v, --version                            version for create

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli virtual-machine instances create [command] --help" for more information about a command.
