Create a Virtual Machine instance

Usage:
  ./cli virtual-machine instances create [flags]

Flags:
      --allocate-fip                       Allocate Fip (default true)
      --availability-zone string           Availability Zone
      --cli.list-links enum[=table]        List all available links for this command (one of "json", "table" or "yaml")
  -h, --help                               help for create
      --image string                       Image
      --key-name string                    Key Name
      --name string                        Name
      --network-interfaces array(object)   Network Interfaces
                                           Use --network-interfaces=help for more details
      --type string                        Type
      --user-data string                   User Data
  -v, --version                            version for create

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
