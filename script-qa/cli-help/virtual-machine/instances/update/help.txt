Update a Virtual Machine instance

Usage:
  ./cli virtual-machine instances update [id] [flags]
  ./cli virtual-machine instances update [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help          help for update
      --id uuid       id
      --status enum   An enumeration. (one of "active" or "shutoff")
  -v, --version       version for update

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --env enum               Environment to use (one of "pre-prod" or "prod") (default "prod")
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use

Use "./cli virtual-machine instances update [command] --help" for more information about a command.
