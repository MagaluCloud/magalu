usage

Usage:
  ./cli virtual-machine usage [flags]
  ./cli virtual-machine usage [command]

Available Commands:
  list        Get usage

Flags:
  -h, --help      help for usage
  -v, --version   version for usage

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli virtual-machine usage [command] --help" for more information about a command.
