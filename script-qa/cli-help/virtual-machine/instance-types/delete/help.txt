Deletes a instance type in DB only

Usage:
  ./cli virtual-machine instance-types delete [instance-type-id] [flags]
  ./cli virtual-machine instance-types delete [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help                    help for delete
      --instance-type-id uuid   instance_type_id
  -v, --version                 version for delete

Global Flags:

      --env enum               Environment to use (one of pre-prod|prod) (default prod)
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use

Use "./cli virtual-machine instance-types delete [command] --help" for more information about a command.
