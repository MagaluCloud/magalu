Delete an object from a bucket

Usage:
  ./cli object-storage objects delete [dst] [flags]
  ./cli object-storage objects delete [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
      --dst string   Path of the object to be deleted
  -h, --help         help for delete

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
      --workers integer        Number of routines that spawn to do parallel operations within object_storage (default 5)

Use "./cli object-storage objects delete [command] --help" for more information about a command.
