download an object from a bucket

Usage:
  ./cli object-storage objects download [src] [dst] [flags]

Flags:
      --continuation-token string   Token of result page to continue from
      --dst string                  Name of the file to be saved
  -h, --help                        help for download
      --max-items integer           Limit of items to be listed (min: 1) (default 1000)
      --src string                  Path of the object to be downloaded

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
      --workers integer        Number of routines that spawn to do parallel operations within object_storage (default 5)
