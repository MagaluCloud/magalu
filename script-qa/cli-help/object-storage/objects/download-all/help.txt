download all objects from a bucket

Usage:
  ./cli object-storage objects download-all [src] [dst] [flags]

Examples:
  ./cli object-storage objects download-all --dst="file1.txt" --max-items=1000 --src="s3://bucket1/file1"

Flags:
      --continuation-token string   Token of result page to continue from
      --dst file                    Name of the file to be saved
      --exclude string              Filename pattern to exclude
  -h, --help                        help for download-all
      --include string              Filename pattern to include (default "*")
      --max-items integer           Limit of items to be listed (min: 1) (default 1000)
      --src uri                     Path of the object to be downloaded

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --region enum            Region to reach the service (one of "br-ne-1", "br-ne-2" or "br-se-1") (default "br-ne-1")
      --server-url uri         Manually specify the server to use
      --workers integer        Number of routines that spawn to do parallel operations within object_storage (min: 1) (default 5)
