Delete an existing Bucket

Usage:
  ./cli object-storage buckets delete [name] [flags]
  ./cli object-storage buckets delete [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help          help for delete
      --name string   Name of the bucket to be deleted

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text
      --region enum            Region to reach the service (one of br-ne-1|br-ne-2|br-se-1) (default br-ne-1)
      --server-url uri         Manually specify the server to use
      --workers integer        Number of routines that spawn to do parallel operations within object_storage (default 5)

Use "./cli object-storage buckets delete [command] --help" for more information about a command.
