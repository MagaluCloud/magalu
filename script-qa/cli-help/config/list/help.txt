List all available Configs

Usage:
  ./cli config list [flags]
  ./cli config list [command]

Available Commands:
  cli.list-links List all available links for this command

Flags:
  -h, --help   help for list

Global Flags:
      --cli.show-cli-globals   Show all CLI global flags on usage text

Use "./cli config list [command] --help" for more information about a command.
